## ✨ What Claw-ED does

Claw-ED is a command-line app that helps teachers create full lesson bundles from one prompt. It builds a set of files you can use right away, with a style that matches your teaching voice.

It is built for:

- Lesson planning
- Curriculum work
- K-12 teaching
- Fast content creation
- Local-first use
- Teachers who want control over the output

## 🧰 What you get

Claw-ED can create a complete 9-file lesson set with:

- Lesson overview
- Learning goals
- Teacher notes
- Student tasks
- Checks for understanding
- Exit ticket
- Materials list
- Differentiation ideas
- Extra support and extension ideas

It uses a large set of tools to shape the output so the bundle feels like it was made for your classroom.
